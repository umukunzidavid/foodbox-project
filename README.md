# foodbox-project

Foodbox Project is a front-end web project for a fresh ingredients and recipe platform. The site is designed to feel modern and engaging, with a homepage that highlights recipes, local suppliers, and an easy way for users to explore meals or sign in.

## Features

- A welcoming landing page with a strong hero section
- Recipe cards for featured meals and trending dishes
- Interactive recipe filtering buttons
- A login portal modal for users
- Additional pages for recipes, sign-up, and admin access

## Project Structure

- food box/food box.html - Main homepage
- food box/recipepage.html - Recipe page
- food box/signup.html - Sign-up page
- food box/adminpage.html - Admin page
- food box/style.css - Main stylesheet
- food box/style2.css - Secondary stylesheet

## How to Run

1. Open the project folder in VS Code.
2. Open any HTML file in a browser, such as food box/food box.html.
3. If you want a smoother preview experience, use the Live Server extension.

## Technologies Used

- HTML
- CSS
- JavaScript

## Notes

This project is a static website prototype focused on design and user experience rather than a full backend system.
